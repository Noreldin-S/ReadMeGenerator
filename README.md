
# ReadMeGenerator
[![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/chong0810/ReadMeGenerator)

# Description

This is a README.md file generator which takes users answers to each question when ran. This app uses a template so that a user can use this README generator app to create a README.md file for any project he or she is working on. This project has taught me how to use node and import different modules.
